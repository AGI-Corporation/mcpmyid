# Agent OS (Activepieces)

**Agent OS** is the foundational operating system for AI Agents, designed to bridge automated workflows with autonomous AI agents. This piece provides a unified toolkit for the Open Agentic Web, powered by research-backed pillars.

## 🚀 Key Pillars

### 1. Cactus-Optimized Execution (Adaptive Routing)
Every tool call is optimized for reliable agentic experiences through features like:
- **Adaptive Repair**: Auto-corrects common LLM mistakes (time formats, etc.).
- **Semantic Guardrails**: Real-time hallucination detection.
- **Deterministic Fallback**: Regex-based extraction to rescue failed tool calls.

### 2. NANDA Protocol Integration (Decentralized Discovery)
Standardized tool discovery for the Agentic Web:
- **AgentFacts (JSON-LD)**: Broadcasts tool capabilities for agent indexing.
- **Standardized Pathing**: Auto-discovery via `/.well-known/agent.json`.
- **Compliance Alignment**: Automatic broadcasting of HIPAA and GDPR flags.

### 3. Virtual Tool Orchestration (Guido Rule Engine)
Build high-level "super-tools" by blending existing pieces:
- **Tool Blending**: Aggregate properties from 280+ pieces into single interfaces.
- **Conditional Validation**: Define logic-based rules for data integrity.

## 🛠️ Usage

### MCP Server Integration
All your Activepieces actions are automatically available as MCP tools. Connect your favorite MCP client (Claude Desktop, Cursor, Windsurf) to start using them.

### NANDA Discovery
Enable NANDA discovery to make your tools discoverable by other autonomous agents via the `/.well-known/agent.json` endpoint.
